# BitPay QR Code

A framework-less QR Code Web Component.

## Usage

```html
<bp-qr-code></bp-qr-code>
```

## Contributing

You'll need [Node.js](https://nodejs.org/en/download/), then:

```bash
npm install
npm start
```

### Production build

```bash
npm run build
```

### Run the tests

```bash
npm test
```
